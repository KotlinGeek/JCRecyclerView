# JCRecyclerView
一个针对 RecyclerView 的极简的加载刷新库。

极简主要体现在对 RecyclerView 进行简单的封装，可以无缝显示加载、刷新的（异常和失败）状态，具有很强的灵活性和实用性，并且不改变 RecyclerView 原有的特性。

JCRecyclerView 只有 200 多行代码就实现了上述功能，并且只有这一个文件。
